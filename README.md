# 야옹 — KBO 야구 응원가·실시간 경기 앱

![야옹 KBO 홈 위젯](https://ya-ong.com/static/landing/og/kbo.png)

야옹은 KBO 중계와 함께 켜두면 현재 타석 선수의 응원 영상과 팀 응원가를
경기 흐름에 맞춰 자동 재생하는 무료 Android·iOS 앱입니다.

## 주요 기능

- KBO 10개 구단 선수 응원가·팀 응원 영상
- 득점·출루·아웃·홈런·공수교대·투수교체 알림 개별 설정
- iPhone Live Activity, Android 알림창 점수판
- 다른 구장까지 볼 수 있는 KBO 홈 위젯
- 전광판·라인업·선수 기록·예매 오픈 정보
